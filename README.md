# Plip Marketing Website

Full-service marketing agency website built with React, Vite, TypeScript, and Tailwind CSS.

## Features

- Modern, responsive design
- Multiple service pages (Solutions, Architecture, Website Services, Paid Advertising)
- Contact form with Netlify Forms integration
- Calendly scheduling integration
- Smooth animations and transitions

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

This site is deployed on Netlify. Push to main branch to automatically deploy.

## Tech Stack

- React 18
- Vite 6
- TypeScript
- Tailwind CSS 4
- React Router 7
- shadcn/ui components
